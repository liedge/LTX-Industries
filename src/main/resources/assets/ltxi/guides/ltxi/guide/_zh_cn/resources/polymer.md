# 聚合物

**聚合物**是一种高强度的轻质材料，其可塑性较强，且可以构成部分 LTX 机器与装备的外壳和内衬。

聚合物产线看上去很复杂，但其实际上完全由可再生的生物材料制成。要合成聚合物，首先需要粉碎甘蔗、竹子等含有树脂成分的植物，此后再用魔能酸提取出树脂单体原料，之后将单体与氧气反应即可得到最终产品。对于各步骤的具体要求，请参考配方查询模组。
