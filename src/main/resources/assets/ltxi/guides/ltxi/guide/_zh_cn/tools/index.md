# ε-系列工具

ε-系列工具是 LTXI 的标准工具产品线。所有工具都无法破坏，由 <Color id="energy">CE能量</Color> 驱动，并且[可升级](../fundamentals/upgrade_system.md)。

## 挖掘工具

### 模块化

<ItemGrid>
<ItemIcon id="epsilon_drill" />
<ItemIcon id="epsilon_shovel" />
<ItemIcon id="epsilon_axe" />
<ItemIcon id="epsilon_hoe" />
</ItemGrid>

钻头、锹、斧和锄是具有*模块化设计*的挖掘工具。其具有 5 种可切换的模式，对应不同的挖掘速度：

- **关闭**：禁止破坏方块，包括创造模式；
- **慢速**：3.0
- **正常**：9.0
- **快速**：20.0
- **超速**：75.0

这些工具的升级系统可以提升工具的挖掘等级（上限）和扩展工具适用的方块种类。锹、斧、锄适合挖掘其对应的方块种类，而为了方便采矿工作，钻头适用于挖掘需要*镐*和*锹*挖掘的方块。
> ε-系列锹的默认升级模块包含精准采集效果。

### 非模块化

<ItemGrid>
<ItemIcon id="epsilon_shears" />
<ItemIcon id="epsilon_wrench" />
</ItemGrid>

扳手和剪刀不具有调整速度和扩展适用方块种类的能力，但仍然支持升级模块。

> ε-系列扳手的默认升级模块能够直接将具有 `ltxi:wrench_breakable` 标签的方块的掉落物吸收进物品栏！ 

## 近战武器

<ItemGrid>
<ItemIcon id="epsilon_sword" />
<ItemIcon id="epsilon_axe" />
</ItemGrid>

剑和斧被设计为*近战武器*，其与[信号系列武器](../weapons/index.md)共享部分升级模块，包括魔咒模块（剃刀和抢夺）和目标筛选器。

> [装备伤害修正](../fundamentals/damage_modifiers.md)系统适用于所有ε-系列近战武器。
