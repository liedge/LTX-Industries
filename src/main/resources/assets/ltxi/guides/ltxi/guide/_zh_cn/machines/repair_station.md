# 修复站

<BlockImage id="repair_station" scale="4" />

我们理解，并不是所有装备都能够使用我们最先进的 ε- 或“秘境”科技，而这些装备只能使用次一级的耐久系统；修复站的出现就是为了解决这类问题。修复站可用使用 <Color id="energy">CE能量</Color> 无限期地将装备的分子结构还原至初始状态，只要工具不损坏，就可以被完美修复至原始状态。

> 具有 `#ltxi:repair_blacklist` 标签的物品和不适用标准耐久机制的物品不可被修复。
