# 地质合成器

<BlockImage id="geo_synthesizer" scale="4" />

## 概述
地质合成器专用于高效制造大量的地质基础材料，也就是石头。

## 配方属性
- 基础工作时间：因配方而异，默认为 3 秒
- 输入：1 种物品，恰好 2 种流体
- 输出：1 种物品
