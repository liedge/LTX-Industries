# 霓虹灯

<GameScene zoom={2}>
<Block id="ltx_lime_neon_light" />
<Block id="energy_blue_neon_light" x="1" />
<Block id="white_neon_light" x="2" />

既美观又节省性能的照明方块，能够发出强度为 **15** 的光照。其可直接在工作台中使用荧石方块合成，但也可以在[材料熔合仓](../machines/gpm/mfc.md)中使用更多种类的发光材料以更高效的配方制作。其中 16 种标准颜色的霓虹灯可以使用相应的染料合成，另外 5 种颜色的则需要使用模组添加的颜料合成。
